A Person-Following Robot

github.com/madebyollin/cse571-racecars

We built a simple person-following robot for CSE 571.

  • The robot uses YOLOv3 for person detection
  • Detections are aggregated across time using area-of-intersection and visual similarity ranking, to avoid swapping people
  • The RealSense Stereo depthmap is used to estimate distance away from the user
